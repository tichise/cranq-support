課題

記録は溜まる。でも、振り返れない。

HealthKit には何百ものワークアウトが眠っている。けれど純正の画面では「今週はどう動けたか」「先月より伸びたか」が見えにくい。Cranq は、その読み解きを引き受けます。

数字の羅列で終わる

距離・時間・カロリーが並ぶだけ。今日のライドが、自分の中で意味を持たない。

続いている実感がない

先週から今週、先月から今月、どれだけ続けられたか。リストを遡らないと分からない。

次の一歩が決められない

「次は何をどう走れば伸びるのか」。判断材料は揃っているのに、自分一人では繋げにくい。

統計を読み解く

ライドもランも、 ひとつの画面で。

HealthKit のワークアウトを Cranq が読み解き、距離・標高・心拍・天気を 1 枚にまとめます。「あの日のライド」が、数字の羅列ではなく物語として立ち上がります。

サイクリング・ランニング・ウォーキング・ヨガを統一表示
ルート・標高・心拍・気温・湿度まで 1 タップで深掘り

走る。振り返る。次へ進む。

GPS で記録し、グラフで振り返り、AI コーチに次の一歩を聞く。3 つのステップが滑らかに繋がります。

STEP 01 · TRACK

GPS で記録する

走ったルート、登った標高、当日の天気まで HealthKit と連動して自動取得。

STEP 02 · REVIEW

グラフで振り返る

週次・月次のチャートで、距離も標高も心拍も一望。続いた日数がそのまま自信になります。

STEP 03 · COACH

AI コーチに相談する

あなたの記録を読んだ AI が、回復・強度・次の練習の指針を日本語で返してくれます。

先週は +18% 走れています。明日は心拍を抑えた 40 分 Z2 で回復を。

続けるための仕掛け

ストリーク、ウィジェット、AI。続けるための装備が揃う。

毎日の小さな一歩を、ホーム画面・通知・AI コーチが後押し。続いていることが、いちばんの励みになります。

ストリーク

連続して動けた日数を記録。0 に戻したくない、という静かな動機が習慣を支えます。

ホーム画面ウィジェット

今週の距離と連続日数をホーム画面に常駐。アプリを開かずに、進捗を一望できます。

AI コーチ

自分の API キー

OpenAI / Anthropic の API キーを自分で持ち込む方式。あなたの記録を読み、回復・強度・次の練習を返します。

安心設計

無料で使えて、データはあなたの端末に。

Cranq は完全無料。サブスクや課金は一切ありません。HealthKit のデータは端末を離れず、AI 機能を使うときだけ自分自身の API キーが裏で働きます。

HealthKit の中だけで完結

ワークアウトデータは HealthKit に留まり、第三者には渡しません。

AI は自分の API キーで動く

OpenAI / Anthropic への通信は端末から直接。Cranq の中継サーバーは存在しません。

アプリ本体は完全無料

サブスクも追加課金もありません。AI を使う場合のみ、自分の API キー側で従量課金が発生します。
