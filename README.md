# Cranq Support

サイクリングやランニングなどの記録を視覚化できるアクティビティ管理アプリ「Cranq」のサポートリポジトリです。

## Cranqについて

CranqはiOS向けのアクティビティ管理アプリです。すべてのデータはHealthKitで管理されるため、他のフィットネスアプリと連携して使うことができます。

### 主な機能

- GPSルートの地図表示・標高チャート・天気情報
- アクティビティの統計・チャート表示
- ストリーク機能によるモチベーション維持
- ホームスクリーンウィジェット
- AIコーチ機能
